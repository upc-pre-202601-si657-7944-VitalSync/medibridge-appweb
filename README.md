# MediBridge App Web

Web clinica de MediBridge construida con React, Vite y TypeScript.

Guia de ejecucion:

```text
docs/GUIA_EJECUCION_MEDIBRIDGE_WEB.md
```

Comandos principales:

```powershell
npm install
npm run dev
npm run lint
npm run build
```

La web consume el backend por API Gateway:

```env
VITE_API_BASE_URL=http://localhost:8080
```
